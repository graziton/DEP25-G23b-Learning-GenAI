How to Use the API

Example Code:


import requests

api_key = 'YOUR_API_KEY'
url = 'https://api.meta-llama.com/v1/query'

headers = {
    'Authorization': f'Bearer {api_key}',
    'Content-Type': 'application/json'
}

data = {
    'prompt': 'Explain basic time management techniques with examples.',
    'model': 'llama-v1'
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

This code sends a request to Meta LLaMA and prints the result. Replace 'YOUR_API_KEY' with your actual key.

Tips, Tricks & Best Practices
