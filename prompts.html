Comprehensive AI Prompts

Housewives

  • I want to cook healthy dinners for a family of four for one week. Provide a daily meal plan (including side dishes) and a consolidated grocery list. We prefer Mediterranean-style meals and need approximately 2,000 calories per day per adult. Include time-saving cooking tips or meal-prep strategies.
  • Create a weekly cleaning schedule for a medium-sized house with three bedrooms and two bathrooms. Break down tasks by day and suggest efficient methods or supplies for each chore.
  • Help me create a simple monthly budget for a family of five with a total monthly income of $5,000. We have the following expenses: mortgage, utilities, groceries, children's extracurriculars, and savings. Categorize the expenses and recommend ways to reduce spending where possible.
  • I am planning a birthday party for my 10-year-old child. Suggest a theme, menu, activities, and a timeline for the event. Also suggest cost-effective decoration ideas and how to keep 10 kids entertained for three hours.
  • Provide a step-by-step guide on basic plumbing maintenance for a home, including how to check for leaks, unclog drains, and safely fix minor issues. Mention necessary tools and any safety precautions.

Engineers

  • Here is a Python function I wrote. It’s meant to parse a JSON file and print specific data fields, but it keeps failing with a TypeError. Please help me identify the bug and optimize the code: [Paste code snippet].
  • I’m designing a microservices architecture for an e-commerce platform. Outline the main components (authentication, catalog, payment, etc.) and suggest suitable communication patterns like REST or message queues. Also propose best practices for data handling to ensure scalability.
  • Explain the concept of concurrency vs. parallelism in simple terms for a non-technical audience. Provide basic examples, real-world analogies, and common scenarios where each is used.
  • I need to design a simple IoT device that measures temperature and humidity and sends data to a central server. Suggest hardware components like sensors and microcontrollers, and outline the communication protocol. Also include an approximate cost estimate.
  • Provide a step-by-step approach to building a linear regression model in Python with scikit-learn. Include data cleaning, feature selection, training, testing, and evaluation metrics. If possible, offer a sample code structure.
  • I’m developing an industrial control system. List the key safety regulations (like ISO or IEC standards) I need to consider for machine operation. Suggest ways to design system redundancies and fail-safes to meet these standards.
